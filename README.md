# Snake Game (Multiplayer)


## Overview
A real-time multiplayer Snake game built with **C#** and **Blazor**. Players connect to a server, control their snake, and compete in a shared world.


## Gameplay
- Collect power-ups to increase your score
- Avoid collisions. Collisions are defined as walls, other snakes and your body.


## Features
- Real-time multiplayer updates
- WASD / Arrow key controls
